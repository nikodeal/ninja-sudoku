Ninja Sudku


    Game Rules

  • The game involves a grid of 81 squares.
  • The grid is divided into nine blocks.
  • Each block has to contain all the numbers 1-9 within its squares.
  • Each number can only appear once in a row, column or box.
  • Every puzzle has just one correct solution.
